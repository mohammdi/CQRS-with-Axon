# CQRS-with-Axon

## Base Project

Generated Spring Boot 3 (Java 21, Maven) project with Axon, H2, and Redis.

### How to run

```bash
cd cqrs-axon
mvn spring-boot:run
```

H2 console: `/h2-console` (JDBC URL: `jdbc:h2:mem:cqrsdb`)

spring boot application with axon fram work design with CQRS &amp; event source
